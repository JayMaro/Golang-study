## Go 특징
- Go는 정적 컴파일 언어(ex. C, C++)
  - 동적 컴파일 언어(ex. Java, Python)
- Go는 강타입 언어(타입을 깐깐하게 본다) 중에서도 타입을 깐깐하게 본다(최강타입 언어...)
- Go는 매우 강력한 가비지 컬렉터가 존재(실행시간 1ms 이하)
- Go는 오픈소스 프로젝트
---
- Go는 증가하는 Stack
  - 메모리가 고갈되지 않는 이상 무한히 증가
- 함수는 미리 선언이 되지 않아도 사용 가능
- Go언어는 함수 호출시 매개변수에 값전달만 지원
